# Capstone Paper

IEEE conference-format paper for the PINN Zimm-Bragg capstone project.

## Files

- `main.tex` — paper source.
- `references.bib` — bibliography.
- `figures/` — figures referenced by `main.tex`. All figures are produced
  programmatically from the project notebooks; do not edit them by hand.

## Compiling

Local LaTeX (TeX Live or MiKTeX with the `IEEEtran` document class):

```
pdflatex main
bibtex   main
pdflatex main
pdflatex main
```

Or, if `latexmk` is available:

```
latexmk -pdf main.tex
```

The IEEEtran class is part of the standard CTAN distribution and does not
need to be downloaded separately.

## Producing the figures

The placeholders referenced from `main.tex` correspond to figures
generated by the project notebooks. To regenerate them all, from the
project root:

| File                           | Generates                                                |
|--------------------------------|----------------------------------------------------------|
| `PINN_PerODE.ipynb`            | `figures/in_domain_fit.pdf`, `figures/scalar_recovery.pdf`, `figures/extrapolation.pdf` |
| `benchmarks.ipynb`             | (numbers for Table I; no standalone figures yet)         |
| `alternative_observables.ipynb`| (supporting numbers for Section V-A)                     |

Run each notebook end-to-end and export the figures it produces to
`paper/figures/` as PDFs.
